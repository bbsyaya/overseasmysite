# overseasmysite 功能说明

基于django框架开发的留学测评网站

分为高中、本科测评两部分

提供美国、英国、加拿大、澳大利亚学校国内外排名

# 开发环境

pycharm + python3 + django 

网址： http://127.0.0.1:8001/newapp/index/ 

服务器线上网址： http://www.deepcomments.com:9200/newapp/index/
